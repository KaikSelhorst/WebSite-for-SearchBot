Ética e Maturidade

  • Não procure por coisas "imaturas" no Searcher sem motivo.
  • Não tente fazer o Searcher exibir conteúdo impróprio nas pesquisas (sujeito a proibição).
  • Não abuse dos bugs encontrados no Searcher, você pode abrir uma solicitação de pull em nosso github.

Peça suporte da maneira certa

  • Ao abrir uma solicitação de emissão / pull no github do Searcher, vá direto ao ponto, para que seja melhor para a equipe entender.
  • De preferência, use o github para relatar bugs, mas se você quiser, pode relatar no servidor de suporte.

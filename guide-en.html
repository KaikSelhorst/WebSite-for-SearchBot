Ethics and Maturity

  • Don't look for "immature" stuff in Searcher for no reason.
  • Do not try to make Searcher display inappropriate content in searches (subject to ban.)
  • Don't abuse the bugs found in Searcher, you can open a pull request on our github.

Ask for support the right way

  • When opening an issue/pull request in Searcher's github, go straight to the point, so it's better for the team to understand.
  • Preferably use github to report bugs, but if you want you can report it on the support server.
